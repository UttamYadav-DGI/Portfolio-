# Uttam Yadav | Full-Stack Portfolio

A high-performance, minimalist, and responsive developer portfolio engineered with **React 19**, **Tailwind CSS**, and **Vite**. This project features a sophisticated "Bento-grid" layout inspired by modern design trends, offering a seamless experience across dark and light modes.

## âœ¨ Key Features

- **Dynamic Multi-Theme**: A robust toggle system between "Deep Obsidian" dark mode and "Paper White" light mode with synchronized CSS variables.
- **Bento Design System**: Organized content sections (Skills, Education, Awards) using a modern grid architecture for maximum visual impact.
- **GitHub Pulse**: Integrated real-time GitHub contribution chart that dynamically adjusts its color palette based on the active theme.
- **Uniform Project Gallery**: A clean, symmetrical 3-column grid showcasing projects with live links, source code, and technical highlights.
- **Micro-Typography**: Utilizing the *Outfit* and *JetBrains Mono* font families for a premium, readable typographic scale.
- **Floating Dock Navigation**: A glassmorphic navigation bar that responds to user scroll and theme changes.
- **Interactive Widgets**: Real-time IST clock, progress indicators for academic status, and verified link buttons for honors/certificates.


